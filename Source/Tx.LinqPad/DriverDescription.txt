<p>Tx is surfacing event sources such as Event Tracing for Windows (ETW) as IObservable sequences. This allows using Reactive Extensions (Rx) and LINQ to Objects to perform queries on:</p>
<ul>
<li>Raw files such as .etl, .evtx and .blg</li>
<li>Real-time sessions (nothing hits the disk)</li>
</ul>
